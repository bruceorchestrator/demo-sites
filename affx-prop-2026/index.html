Scope of Work

4 модуля

🤖 2 недели

Module 1: AI Sales Agent

$2,000

Telegram бот-квалификатор — мгновенная обработка входящих лидов 24/7. Контекстный диалог: определяет уровень, интересы, бюджет. Автоматический handoff на менеджера для оплат.

Deliverables

  • ✓ Работающий Telegram бот
  • ✓ Интеграция с CRM
  • ✓ Сценарий диалогов
  • ✓ Дашборд метрик

Ценность

7,000 лидов сейчас без ответа. Если бот конвертирует хотя бы 1% из них — это 70 продаж. При среднем чеке курса — окупается в первый месяц.

Замена: наём менеджера на обработку лидов ~$800-1,200/мес

🎯 2-3 недели

Module 2: AI Nurturing System

$1,500

Behavior-driven follow-up: автоматические касания на основе действий лида. 3-4 касания за 2-3 дня. Сегментация: записался но не пришёл / был но не купил / начал но бросил.

Deliverables

  • ✓ Nurturing flows для каждого сегмента
  • ✓ Аналитика open/click/conversion
  • ✓ Шаблоны сообщений
  • ✓ A/B тестирование

Ценность

4,000 лидов с вебинара → 20 оплат (0.5%). С nurturing конверсия растёт до 1.5-2% — это 60-80 оплат вместо 20. x3 выручка с каждого вебинара.

Экономия: ~40 часов/мес ручных follow-up менеджерами

🎬 3-4 недели

Module 3: AI Content Machine

$1,500

Один вебинар (1.5ч) → 20-30 единиц контента автоматически. Reels скрипты, карусели, TG посты, SEO-статьи. Programmatic video с субтитрами и брендингом.

Deliverables

  • ✓ Контент-пайплайн
  • ✓ Первые 50 единиц контента
  • ✓ Шаблоны по форматам
  • ✓ Видео-генерация (Reels)

Ценность

Сейчас 2-3 поста/нед, с AI — 15-20/нед. Контент-менеджер + копирайтер = $1,500-2,000/мес. AI делает x5 больше за $0 операционных расходов.

Экономия: ~$1,500/мес на контент-команду

🔍 4 недели

Module 4: SEO Infrastructure

$1,200

SSR migration для Nuxt (чтобы Google видел сайт). AI-генерация SEO-статей из вебинаров. Internal linking + CTA на курс в каждой статье.

Deliverables

  • ✓ SSR migration
  • ✓ 20 SEO-оптимизированных статей
  • ✓ Keyword strategy
  • ✓ Meta tags + internal linking

Ценность

Сейчас 0 органического трафика. Каждая SEO-статья = входная точка в воронку. 20 статей за 4 недели — через 3-4 мес это бесплатный трафик навсегда.

Замена: SEO-агентство $1,000-2,000/мес + копирайтер $500-800/мес
